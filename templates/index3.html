Utiliser

Second projet : site web de visualisation

Réaliser une application/site web capable d'extraire des informations d'une base de données sqlite et d'en faire des charts

Durant le premier projet, on nous a proposé d'écrire nos propres exercices en python pour la plate-forme INGInious qui est déployée à l’UCLouvain, et où des des dizaines de milliers de soumissions sont faites chaque semaine. Le but de ce projet est de synthétiser les soumission qui y sont faites grâce à un application de notre qui :

  • Fonctionne grâce au framework Flask
  • Synthétise les données de manière pertinente
  • Affiche des diagrammes Chart.js
